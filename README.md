🎧 MoodTUNES-AI-Powered-Mood-Based-Music-Recommendation-System

An AI-Powered Human-Centric Music Recommender using NLP + Facial Emotion Recognition + Explainable AI


🔥 Project Overview
This project is an AI-driven music recommendation system that understands the user's mood from:

💬 Typed text — describe how your day was (NLP sentiment analysis)
📷 Facial emotion analysis — live webcam or uploaded photo (DeepFace)
🌦️ Live city weather — mood mapped from real-time weather conditions
📂 Image upload — drag & drop any face photo for emotion detection

Based on the detected emotion, the system recommends songs that match, support, or improve your mood.
Every suggested song comes with explainable reasoning and direct ▶️ Spotify and ▶️ YouTube play buttons.
